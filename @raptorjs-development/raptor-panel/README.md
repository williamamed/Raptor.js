Raptor Panel
=======
Componente de proyecto de Raptor.js que actual como portal de desarrollo, solo está habilitado en modo desarrollo vía cli y provee todas las funciones
para integrar funcionalidades utilitarias, se encuentra accesible a través de la ruta http://localhost:4440/raptor.

Contiene las funcionalidades para configurar el framework, gestionar los componentes de forma visual, publicar recursos y gestionar modelos (proximamente este será
movido hacia el componente @raptorjs/orm)
